#wsBackend-Fabrica26.1

o projeto recebe informações de usuario e os cadastra para um banco de dados /admin, podendo ser mostrado em project/

As dependencias necessarias estão no arquivo requirements.txt

para acessar a aba admin/, o login e senha são ambos "admin", você poderá criar um cadastro contendo algumas informações, e então poderá ir para a aba project/, onde irão mostrar os dados cadastrados
